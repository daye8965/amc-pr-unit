markdown# 언론유닛 홍보 실적 대시보드

서울아산병원 언론유닛 2021–2026 홍보 실적 인터랙티브 대시보드

---

## 사용 방법

1. `언론유닛_홍보_대시보드.html` 파일을 다운로드
2. 브라우저(Chrome 권장)에서 열기
3. 탭 전환으로 각 분석 화면 이동
4. PDF 저장 버튼으로 보고서 출력

---

## 탭 구성

| 탭 | 내용 |
|---|---|
| 📊 전체 현황 | 누적 건수, 구분별·연도별 차트 |
| 📈 연도별 비교 | 2021~2026 추이 분석 |
| 🔆 2026년 상세 | 월별·담당자·진료과·교수별 현황 |
| 🏥 진료과 분석 | 진료과·교수별 홍보 건수, 소외 진료과 |
| 👥 담당자 현황 | 담당자·연도 필터 + 전체 목록 |
| 🔍 키워드·브랜딩 | 빈출 용어, 클러스터, 브랜딩 인사이트 |
